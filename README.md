# Arthera White Paper UI

- Web app: https://whitepaper.arthera.net/
- Contract: https://github.com/artheranet/whitepaper-nft-contracts

## Install

```bash
pnpm install
```

## Run

Create a `.env` on the model of `.env.template`:

```sh
cp .env.template .env
```

Add your own keys in your `.env` file, then:

```bash
pnpm dev
```

## Build

```bash
pnpm build
```
